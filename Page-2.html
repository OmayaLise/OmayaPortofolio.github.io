PotatOS

Custom Game Engine
Group project in C++

Dowload

PotatOS is a custom game engine inspired by Unity. It is a 3 persons project made in 4 months in C++.

Acessibility features
Features
  • Physic system
  • Components system (collider, lights, meshes, audiosource,etc)
  • Guizmo
  • Reflection on variables
  • Sound system with mixer 
  • 3D FBX models availables
  • Save and load system for the scenes 
  • UI (hierarchy , component and files windows) customizable
  • Several viewmodes to help the developer vizualize a color-blind person sight
  • Integrated and fully customizable subtitles system
  • Integrated and fully customizable input map
