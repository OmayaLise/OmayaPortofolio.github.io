PotatOS Engine 

Description

  • Student project realized by a group of three
  • C++ 
  • Made for PC 

Features

  • Create modify and save a scene 
  • Create and modify a gameobject and add component to it (light, canva, animation, audiosound etc)
  • Add and use assets (models with FBX, animation and sounds)
  • Modify audio mixer
  • Modify controls and camera in game and in editor
  • Activate different color-blind modes 
  • Create scripts
